Description:
The Hospital Patient Record System is a console-based C project that is designed to efficiently manage and record patient informations in a hospital. It allows hospital staffs to add, search, update and delete patient records using the console. The program ensures record management and storage using file handling. It replaces old paper based system with  faster and reliable digital solution.

Main Features:
Add patient info (Id, name, age, blood group, disease, doctor name, admission date).


Search and display patient details using Id.


Update and delete existing patient records.


Display all patient details in a structured format.


Save and retrieve patient data from a saved file.
